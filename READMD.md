## ✨ 功能特色

- ✅ 支援 PDF 與圖片雙模式切換
- 📄 自動偵測圖片頁數，無需手動設定
- 🔘 上一篇 / 下一篇文章切換
- 📷 圖片模式下顯示「第 X 頁 / 共 Y 頁」
- 🌐 可部署至 GitHub Pages 作為個人教材網站
